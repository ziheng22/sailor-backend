# 启航手册

sailor工作室是一个综合型实践团队，成员由大学老师和学生构成。我们关注真实软件项目中的需求理解、产品设计、工程实现、测试交付和持续维护。

## 方向

- Web 前端与全栈应用
- AI 应用与知识库系统
- 数据分析与可视化
- 校园信息化系统
- 科研原型与竞赛作品

## 培养方式

我们希望每个成员都能经历从“会写功能”到“能交付系统”的完整过程。工作室会通过项目制协作、代码评审、技术分享和阶段复盘帮助成员积累工程经验。

## 对外合作

我们欢迎课程建设、科研项目、校园系统、企业原型和竞赛项目合作。第一版网站中的内容均为占位，后续会逐步替换为真实案例。
